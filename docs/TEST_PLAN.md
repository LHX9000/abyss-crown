# ABYSS CROWN 测试文档

本文件用于归档 `ABYSS CROWN` 的测试内容。当前项目已经完成第一版 C++/SDL2 实现，本测试文档记录需要覆盖的测试点和已执行的基础验证。

- SDL2 窗口启动测试。
- 键盘输入测试。
- 地图碰撞测试。

- 胜利流程测试。

## 2. 测试方式

主要通过手动运行和观察完成：

- 使用 `make` 编译项目。
- 使用 `make run` 启动游戏。
- 在 SDL2 窗口中操作角色并观察结果。

## 3. 已执行测试

### 3.1 SDL2 环境检查

执行内容：

```bash
sdl2-config --version
```

结果：

- 已检测到 SDL2 `2.32.10`。
- 已检测到 SDL2_ttf `2.24.0`。
- 本机 C++/SDL2 编译环境可用。

### 3.2 编译测试

执行内容：

```bash
make
```

结果：

- 编译成功。
- 已生成 `build/abyss_crown`。
- 二次执行 `make` 时显示无需重编译，说明构建依赖关系正常。
- 已成功链接 `SDL2_ttf`。
- 已在 `feature/end-screen-polish` 分支完成结束页接口改造并通过全量编译。

### 3.3 SDL2 启动检查

执行内容：

```bash
./build/abyss_crown
```

结果：

- 程序成功进入 SDL2 运行状态。
- 中文字体、标题背景图和 SDL2_ttf 均成功加载，没有输出资源加载警告。
- 中文主菜单、游戏说明场景、三个菜单按钮和返回主页动作已通过代码级检查。
- 后续仍建议由用户在本机窗口中进行完整人工游玩测试，包括移动、拾取、胜利和失败流程。
